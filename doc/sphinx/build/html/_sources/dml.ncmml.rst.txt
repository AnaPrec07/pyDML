Nearest Class Mean Metric Learning (NCMML)
==========================================

A distance metric learning algorithm for nearest class mean (NCM) classification. It learns a transformation that optimizes the NCM expected score.

Watch the full NCMML documentation `here <dml.html#module-dml.ncmml>`_.

References
----------

Thomas Mensink et al. “Metric learning for large scale image classification: Generalizing to new
classes at near-zero cost”. In: Computer Vision–ECCV 2012. Springer, 2012, pages 488-501.
